Squaring a number quickly...
Intro
Squaring a number is a frequently used operation. Learning how to do it quickly is a good advantage to have. We have divided this trick into 3 sub-tricks. The first sub-trick is used to find the square of any number with a unit digit 5 i.e. 5, 15, 35, 105, 1365 etc. The second sub-trick can be used to find the square of a number that is closer to a base. (If you don't know what a base is, then please go through our tutorial here.) The third sub-trick can be used for any number. So, let's start learning.
Sub-trick 1

This trick is used to find the square of any number with unit digit 5 i.e. 5, 35, 175, 1675 etc. Treat the number you are given as 'x5'. I mean, if number is 15, then x is 1. If number is 35, then x is 3. If the number is 175, then x is 17. Similarly when the number is 1675, then x is 167. Now that you have your x, calculate 'x * (x+1)' i.e increment x by 1 and multiply it with x. For ex, if your number is 15, then x is 1 and 'x * (x+1)' is 2. Now write 'x * (x+1)' with 25 at the end. That's it, you have your answer. Go through the examples if you didn't understand the trick properly.

Example 1

Let the number be 35. Then x will be 3. 'x + 1' will be 4. 'x * (x + 1)' will be 12. Write 12 and then 25 i.e. 1225. That's it, the answer is 1225. The square of 35 is 1225. Use the calculator in the header to check if it's correct.

Math Behind 'Sub-trick 1'

We assumed the number to be 'x5'. In algebra, x5 = (x * 10) + 5

For example, 175 = x5 implies x = 17. 175 = (17 * 10) + 5 = (x * 10) + 5

Number2 = ((x * 10) + 5)2
              = (x * 10)2 + 2 * (x * 10) * 5 + 25
              = 100 * x2 + 100 * x + 25

Number2 = 100 * x * (x + 1) + 25

'100 * x * (x + 1) + 25' is same as writing '25' at the end of 'x * (x + 1)'.

Therefore, square of Number is same as writing '25' at the end of 'x * (x + 1)'.

Sub-trick 2

This sub-trick is useful for finding the square of a number which is closer to a base. For example, 97 is closer to 100, 1007 is closer to 1000, 14 is closer to 10 and so on.

Step 1

For a given number, find the nearest base. Also find the deviation of the number from the base. If you don't know or don't remember bases and calculating deviations, then jump to our tutorial here.

Step 2

Now, you have a number and a deviation from the nearest base. Add the number and the deviation. Let this sum be 'x'. Multiply the deviation with itself i.e. square it. Let this number be 'y'.(If the number is closer to a base, then the deviation, generally, will be less than 20. If not, treat 'squaring the deviation' as a new problem or use 'Quick Multiplication' trick to find the square.). The answer will be equal to '(x * base) + y'. Note that 'x * base' is same as writing the number of zeroes of base at the end of 'x'. If you didn't understand the trick properly, go through the examples below.

Example 1

Let the number be 103. The nearest base to 103 is 100. Deviation of 103 from 100 is 103 - 100 = 3.

Number = 103
Base = 100
Deviation = 3
x = Number + Deviation = 106
x * Base = 10600
y = Square of Deviation = Square of 3 = 9.
Answer = 10609.

Math Behind 'Sub-trick 2'

Let the number be 'n', the base be 'b', the deviation be 'd'.

Deviation = Number - Base
Number = Base + Deviation
n = b + d

n2 = (b + d)2
    = b2 + 2 * b * d + d2
    = b * (b + 2 * d) + d2
    = b * (b + d + d) + d2
    = b * (n + d) + d2

We assumed 'n + d' to be 'x'.
We also assumed d2 to be 'y'

Therefore Number2 = x * Base + y

Sub-trick 3

Let's define a new term 'Pseudo Base' ('pseudo' means false, pronounced as 'seudo'). Pseudo base is multiple of a base. For example 400 is multiple of 100 and hence is a Psuedo Base. We use this trick when the number is not closer to any base in particular but closer to a Pseudo Base.

Step 1

Once you have a number, find a Pseudo Base that is closest to the given number. For example, if the number is 487, the Pseudo Base is 500, if the number is 893, the Pseudo Base is 900 and if the number is 71, the Pseudo Base is 70 and so on. Also, find the deviation of number from the Pseudo Base. Please note that for a number, there might exist more than one Pseudo Base. In such cases, we choose the pseudo base which simplifies our calculation. For example, if the number is 513, choosing 500 as Pseudo Base gives us much simpler calculation than choosing 600 as Pseudo Base

Step 2

Add the number and deviation. Let the result be 'x'. Multiply the deviation with itself i.e. square it. Let this number be 'y'.(If the number is closer to a base, then the deviation, generally, will be less than 20. If not, treat 'squaring the deviation' as a new problem or use 'Quick Multiplication' trick to find the square.). The answer will be equal to '(x * Pseudo Base) + y' . If you didn't understand the trick properly, don't worry and follow the examples given below.

Example 1

Let the number be 71. The nearest Pseudo Base to 71 is 70. Deviation of 71 from 70 is 71 - 70 = 1.

Number = 71
Pseudo Base = 70
Deviation = 1
x = Number + Deviation = 72
x * Pseudo Base = 72 * 70 = (72 * 7) * 10 = 5040
y = Square of Deviation = Square of 1 = 1.
Answer = 5041.
