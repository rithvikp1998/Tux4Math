Bases and Deviations
Intro

A base is anything that is power of 10. For example, 10, 100, 1000, 10000 etc. are all bases, whereas 45, 52, 4560, 56600, 404 etc. are not. The deviation of a number from a base is equal to the difference of number from the base. For example, the deviation of number 14 from base 10 is 14 - 10 = 4. Please note that deviations can be negative also. For example, the deviation of number 96 from base 100 is 96 - 100 = -4.

Deviation = Number - Base
