About

Tux4Math is an open source web app designed to teach kids speedy math tricks. You can learn math tricks and also practice them. This web app is built by Tux4Kids, an organisation which designs and develops educational softwares. We hope you enjoy learning with our website. Thank You!

Report Bugs

Tux4Math is an open-source web app. If you find any bugs, you can fix them yourselves! Cool, isn't it? Or you can report them to me. Thank You!
